Insert Mode

User clicked Insert and armed a component. Crosshair cursor, teal drop-line with inward arrow end-caps (>———<), and a floating Place: Button pill follow the cursor. Naming — the Select-mode button is Replace, the Insert-mode button is Place. Both open the same sidebar tab.
