Approach 3 — Command Bar (Spotlight/Palette)

A persistent command bar at the top (like VS Code's Command Palette, but always visible as a single-line input). Typing filters to actions, components, or sends a message to the AI. Empty → shows default property editors. Extremely fast for power users — type what you want, hit Enter.

A · Empty bar — element selected

When nothing is typed, the panel functions like the current Design tab with property editors. The command bar waits at the top.
