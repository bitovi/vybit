Approach 4 — Smart Tabs + Persistent Message Bar

Keep tabs (they're a good navigation paradigm) but rename them to Edit and Add to match user intent. Add a persistent message bar visible on all tabs — the most common cross-cutting action is always one keystroke away. Tabs adapt based on whether an element is selected.

A · Edit tab — element selected

The Edit tab is the default when an element is selected. Shows the property editors (current Design tab) with a persistent message bar above. "More actions" at the bottom.
