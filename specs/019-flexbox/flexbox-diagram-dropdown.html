Flexbox Diagram Dropdown — Prototype

Mini flex diagrams in a dropdown triggered by clicking a pill chip — like ScaleScrubber but with spatial diagrams instead of a list. The pill shows the current value plus an arrow indicating which axis it controls. Arrow direction updates when flex-direction changes.

Pill anatomy:
[axis-arrow] value-name [▾]
  • axis-arrow — shows which axis this property controls. Updates with direction.
  • value-name — the current Tailwind class (short form)
  • ▾ caret — hints that this is a dropdown, rotates when open

Arrow logic by direction:
  • flex-row → justify = →, align = ↓
  • flex-col → justify = ↓, align = →
  • flex-row-reverse → justify = ←, align = ↓
  • flex-col-reverse → justify = ↑, align = →
