# Bitovi Brand Assets

This document describes official Bitovi brand assets. The source assets live at:
`https://github.com/bitovi/vybit/tree/bitovi/context/bitovi/brand-assets`

---

## Logo Variants

The Bitovi wordmark logo is available in three color schemes:

### Dark Green (Primary)
- Files: `Logo-dark green.svg` / `.png` / `.eps`
- Color: Dark teal/green `#00333D`
- **Use on**: Light/white backgrounds
- **Primary brand identifier** — use as default

### Red-Orange
- Files: `Logo-red-orange.svg` / `.png`
- Color: BitOrange `#F5532D`
- **Use for**: Accent or variety on light backgrounds

### White
- Files: `Logo-white.svg` / `.png` / `.eps`
- **Use on**: Dark backgrounds, teal backgrounds, or over photos

---

## Mascot Variants

The Bitovi mascot is a friendly character with headphones, representing approachability and technical expertise.

### 2D Versions
- `Mascot-2D dark green.eps`
- `Mascot-black.svg` / `.png`
- `Mascot-dark green.svg` / `.png`

### 3D Versions
- `Mascot-3D large uses.eps` — Optimized for larger applications
- `Mascot-3D small uses.eps` — Optimized for smaller sizes

---

## File Formats

| Format | Best For |
|--------|----------|
| SVG    | Web use (scalable, recommended for UI) |
| PNG    | Raster with transparency |
| EPS    | Print and professional design software |

---

## Usage Guidelines

- Use the **dark green logo** as the primary brand identifier
- Use the **white logo** on dark/teal backgrounds for contrast
- The **mascot** represents approachability and technical expertise
- Maintain adequate **clear space** around logos and mascot
- **Do not** alter colors, proportions, or add effects to brand assets
