Brand Guideline — BitOrange (#F5532D) should be used sparingly as an accent: selected states, active nav, category labels, "bits" decorations. Teal is the workhorse for interactive elements, focus states, and current-value indicators.
Typography
Three-font system: Poppins Bold for headings · Inter for all UI text · Roboto Mono for class names & values.
